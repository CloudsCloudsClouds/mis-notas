# Funciones-L

Funciones especiales que codifican informacion espectral o aritmetica sobre objetos tales como numeros, campos, curvas o simetrias

Funcionan tambien como una especie de "firma" o de forma que pueden tener distintas estructuras matematicas.

Menciones celebres:
- Funcion Riemann Zeta
- Funciones L de Dirichlet
- Funcion L de Curva Eliptica
